My projects


Spare Part Manager

December 2017 - January 2018

This is the final project I created at the end of seven week intense Java Course. The idea is to aid in managing spare part movements between a number of central IT hubs and a number of remote locations (for example: Vessels in the middle of the Ocean or Camp in the middle of the Desert).

I deployed my application on Heroku, so you can login and have a look. Quick instruction on how to get the most from your experience:

  • Use existing credentials I prepared for you: admin1 (password: admin1), user1 (password: user1), user2 (password: user2).
  • Registration is currently disabled for demo version, but it in production it will be fully functional.
  • Database is already preloaded with some data, it is fully functional , so you can add items as well.
  • Try loggin in as admin and regular user: their views are quite different.
  • Admin can see users, locations, manufacturers and spare part catalogs. Also spare parts in all locations and all shipments are visible.
  • Regular user is asigned to a particular location and can only see items in this location plus all shipments to/from his location.
  • Main part of the website is the 'Spare Part' section. various groups of parts (grey buttons) have different actions available. Play around sending parts to various locations, receiving them, placing in the system and returning to global hubs.
  • Note: some of the options (user editing, items deleting) are disabled for demo version.
